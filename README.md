# 📌 Prompt Management System

## 📖 Description
Le projet **Prompt Management System** est une application web permettant de gérer, organiser et utiliser des prompts (instructions) de manière efficace.  
Elle permet aux utilisateurs d'ajouter, modifier, supprimer et rechercher des prompts selon leurs besoins.

---

## 🎯 Objectifs
- Faciliter la gestion des prompts
- Améliorer la productivité des utilisateurs
- Offrir une interface simple et intuitive
- Permettre le filtrage et la recherche rapide

---

## 🛠️ Technologies utilisées
- **Frontend** : HTML, CSS, JavaScript  
- **Backend** : PHP  
- **Base de données** : MySQL  
- **Serveur** : Apache / Nginx  

---

## ⚙️ Fonctionnalités principales
- 🔐 Authentification (login / logout)
- ➕ Ajouter un prompt
- ✏️ Modifier un prompt
- ❌ Supprimer un prompt
- 🔍 Rechercher un prompt
- 📊 Tableau de bord (admin)

---

## 🧑‍💻 Rôles utilisateurs
- **Admin**
  - Gère les utilisateurs
  - Accès complet au système
- **Développeur**
  - Ajoute et utilise les prompts
