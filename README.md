# Prompt Repository - Knowledge Base des Prompts Performants

## Contexte
Dans le cadre de DevGenius Solutions, les développeurs utilisent quotidiennement des LLM (Large Language Models) pour générer du code, créer des scripts ou rédiger de la documentation technique.  
Les prompts performants se perdent souvent dans les historiques de chat, rendant leur réutilisation difficile.  

**Objectif du projet :** Créer une plateforme interne pour stocker, gérer et réutiliser ces prompts de manière organisée et sécurisée.

---

## Rôles & Fonctionnalités

### Admin
- Gérer les catégories de prompts (ajouter, éditer, supprimer).  
- Voir les contributeurs les plus actifs (statistiques sur les prompts créés).  

### Développeur
- Créer de nouveaux prompts.  
- Éditer ses propres prompts.  
- Supprimer ses prompts.  
- Filtrer les prompts par catégorie.  

---

## Objectifs
- Conception d’une base de données relationnelle sécurisée (MySQL).  
- Gestion des utilisateurs avec authentication (PHP & sessions).  
- Implémentation complète du cycle CRUD pour les prompts.  
- Interface simple et responsive pour dashboard admin et développeur.
